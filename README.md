# AI Impact on Students - Performance Predictor

This is a Streamlit application designed to predict a student's **Post-Semester GPA** based on their GenAI tool behaviors, academic history, and traditional study habits.

## Dataset
The app automatically pulls down the latest dataset from Kaggle using `kagglehub`:
* **Dataset:** `laveshjadon/ai-impact-on-students`
